# منصة قسم المسار الأدبي 🎓

منصة إلكترونية كاملة لإدارة قسم المسار الأدبي — مدرسة النهضة الابتدائية للبنات.
مبنية بـ **Expo (React Native)** — تعمل كموقع ويب + تطبيق أندرويد + آيفون من نفس الكود، وقاعدة البيانات **Convex** سحابية لحظية.

## الدخول
- المستخدم: `admin`
- كلمة المرور: `admin123`

## الأقسام
لوحة معلومات، المعلمات، الطالبات (10 شعب)، الاجتماعات (جماعي/فردي)، الزيارات الصفية، التقرير الدوري، التقييم السنوي (حساب تلقائي من 100)، التحصيل الأكاديمي (رسوم بيانية)، الخطة الفصلية، الأعمال الكتابية، التطوير المهني والقراءة المهنية، التقرير الشهري للمنسقة، الإنجازات، متابعة التوصيات، ولوحة تحكم.
كل استمارة تُطبع PDF بتنسيق رسمي مطابق للنماذج الورقية.

## التشغيل
```bash
cd platform
npm install

# 1) ربط قاعدة البيانات Convex (مرة واحدة فقط — يفتح المتصفح لتسجيل الدخول بحسابك)
npx convex dev
#    اختر المشروع الموجود successful-panda-576 ليتم رفع الدوال (schema + functions)

# 2) تشغيل الويب
npx expo start --web

# 3) تشغيل على الجوال: ثبّت تطبيق Expo Go وامسح QR
npx expo start
```

## البيانات الأولية
عند أول تشغيل تُستورد تلقائياً: 12 معلمة، 187 طالبة، 10 شعب (من ملفات القسم الأصلية).
يمكن إعادة الاستيراد من «لوحة التحكم».

## النشر
- **الويب**: `npx expo export --platform web` ثم ارفع مجلد `dist` على Vercel/Netlify.
- **أندرويد/آيفون**: `npx eas build` (يتطلب حساب Expo).
